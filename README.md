# ML Models
This repository showcases a small collection of Machine Learning projects I have completed as part of my university coursework, displaying my experience with data preprocessing, exploratory data analysis (EDA), model training, evaluation, and interpreting results.

## Projects
### 🏘️ Project 1: Airbnb Price Predictions — Regression Models
I built and compared regression models to predict Airbnb listing prices. This project focuses on feature preparation, exploratory analysis, model evaluation, and comparing baseline and regularised regression methods.

**💡 Main techniques used:**  
Linear Regression, Ridge Regression, Lasso Regression, feature scaling, RMSE, R². 

---

### 🔥 Project 2: Wildfire Intensity Levels — Classification
I developed classification models to predict wildfire intensity levels from structured data. This project focuses on supervised classification, preprocessing pipelines, hyperparameter tuning, and evaluating model performance across multiple classes.

**💡 Main techniques used:**  
Decision Tree, Support Vector Machine, MLP Classifier, GridSearchCV, confusion matrices, classification metrics

---

### 🧬 Project 3: Cancer Image Classification
I built machine learning models for low-resolution histopathology image classification. This project compares traditional machine learning approaches with CNN-based models for cancer detection and cell-type classification.

**💡 Main techniques used:**  
Image preprocessing, CNNs, data augmentation, class weighting, threshold tuning, accuracy, macro F1-score
